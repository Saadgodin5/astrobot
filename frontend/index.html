About the Project

A Graduation Project

AstroBot is a university graduation project built by four students passionate about AI, automation, and clean engineering. It combines a modern web stack (Node.js, Express, PostgreSQL) with an n8n automation backbone and Stable Diffusion for image generation — all wrapped in a space-themed conversational UI.
